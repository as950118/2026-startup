## 3. 성장 동인

- **디지털 전환**: 온라인 B2B 플랫폼, 실시간 재고·가격, 자동화 발주
- **투명성·이력추적**: 원산지·식안전·규제(FDA 등) 수요
- **기술 도입**: AI, 블록체인, IoT — 물류·품질·수요 예측
- **지속가능·로컬 소싱**: 윤리·지역 농산물 수요
- **콜드체인·물류 최적화**: 냉장冷链 비용·폐기 감소

## 4. 국내 시장 (한국)

| 항목 | 수치·내용 |
|------|------------|
| **식자재 유통 시장** | 2015년 37조원 → 2020년 55조원 → 2025년 **64조원** 전망 |
| **구성** | 시장의 80%+ 영세업체, 대부분 아날로그 운영 |
| **농축수산물 온라인** | 2024년 7월까지 연 7.3조원+, 월평균 1조원 돌파; 연간 12조원+ 전망(전체 농축수산물 70조원의 약 17%) |
| **농수산물 온라인도매** | 2025년 거래실적 **1조원** 달성 목표·실적 |
| **축산물 B2B·온라인도매** | 2024년 목표 **2,000억원**; 미트박스 등 플랫폼 성장 |
| **효과** | 온라인 직거래로 도매 경유 대비 출하·도매 비용 약 **10% 절감** |

## 5. AI·기술 적용 영역 (시장 관점)

| 영역 | 적용 내용 |
|------|-----------|
| **수요·재고 예측** | 농산물·식자재 단위 수요 예측, 자동 발주·재고 최적화(Church Brothers 사례: 단기 예측 정확도 40% 개선) |
| **품질·신선도** | 이미지·분광·센서 + AI로 신선도·유통기한 예측, 90%+ 분류 정확도 연구 |
| **냉장 물류** | 수요 예측(에러율 5.28% 등), 창고 배치·출발 시각 최적화, 온도 예측으로 손실 방지 |
| **이력·투명성** | 블록체인·IoT·AI 연동, FDA 등 규제 대응 |
| **플랫폼 AI** | 추천·검색·오탈자 정정·개인화(미트박스), 음성·이메일 기반 발주(GrubMarket) |
