# AI 기반 육류·식자재 유통 플랫폼 정리

> **목적**: AI 기반 육류·식자재 유통 플랫폼 시장·트렌드·경쟁력 분야를 한곳에 정리. 창업·투자 검토용.  
> **출처**: 시장 리포트, 경쟁사·유스케이스 조사, 사업화 계획서, 조사 출처 문서

---

## 1. 방향 요약

| 구분 | 내용 |
|------|------|
| **정의** | AI(추천·수요예측·재고·품질·이력추적 등)를 활용한 육류·식자재 B2B 유통·공급망 플랫폼 |
| **글로벌 B2B 식자재 마켓** | 2022~2024년 약 290~480억 달러 구간, 2030년 810억~2,040억 달러 전망(CAGR 5.8~20% 리포트별 상이) |
| **국내 식자재 유통** | 2020년 55조원 → 2025년 64조원 전망; 온라인 농축수산물 월 1조원+ (2024), 축산물 B2B 2,000억원급 목표 |
| **전략** | 육류·식자재 **니치 1개**(도매·가공·냉장물류·품질·이력 등) PMF 후 AI 고도화·확장 |

---

## 2. 경쟁력을 가질 수 있는 분야

| 분야 | 설명 | 포지션 |
|------|------|--------|
| **AI 추천·개인화** | 구매 이력·상품 속성·행동 데이터 기반 추천, 오탈자 정정·선호도 정렬(미트박스 등) | B2B 이커머스 플랫폼 차별화 |
| **수요·재고 예측** | SKU·계절·날씨·이벤트 기반 수요 예측, 자동 발주·재고 최적화(GrubMarket, 토더) | 유통사·식당·가공사 비용·폐기 절감 |
| **육류 가공 최적화** | 도체 정렬·생산 스케줄·커팅 플랜 AI로 가치 극대화(Völur–JBS) | 도축·가공사 마진·이용률 개선 |
| **품질·신선도 AI** | 이미지·분광·센서 기반 신선도·유통기한 예측, 이력추적 연동 | 유통·소비자 신뢰·폐기 감소 |
| **냉장冷链·물류 AI** | 수요 예측·창고 배치·경로 최적화, 예측적 온도 모니터링 | 냉장 물류 비용·손실 감소 |
| **이력추적·투명성** | 블록체인·IoT·AI 연동 farm-to-fork 추적, 규제 대응(FDA 등) | 식안전·규제·프리미엄 브랜딩 |
| **니치 세그먼트** | 육류 전용·소포장 직배송·외식 프랜차이즈 전용·지역/품목 집중 | 기존 종합 플랫폼과 구분 |

**정리**: “플랫폼만”이 아니라 **AI 레이어(추천·예측·품질·이력·물류)**에서 차별화할 수 있음. 데이터 축적이 핵심.

---

## 3. 피할 것·차별화 포인트

- **피할 것**: 단순 마켓플레이스 중복(가격·상품 수만으로 경쟁); AI 없이 “디지털 전환”만 강조; 규제·식품안전 무시.
- **차별화**: **데이터 기반 AI**(구매·재고·품질·이력)·**니치 1개**(육류·소포장·프랜차이즈 등)·**단위 경제**(수수료·구독·SaaS)·규제·콜드체인 연동.

## 4. 시장·트렌드 요약

| 항목 | 내용 |
|------|------|
| **국내** | 식자재 64조원(2025), 80%+ 영세·아날로그; 농축수산물 온라인 월 1조원+, 축산 B2B 2,000억원 목표 |
| **글로벌 B2B 푸드 마켓** | 2024년 82억~476억 달러 구간(리포트별 상이), 2030~2034년 810억~2,040억 달러, CAGR 5.8~20% |
| **AI 적용** | 재고·발주 자동화, 수요 예측, 품질·신선도 예측, 이력추적·블록체인, 냉장 물류 최적화 |
| **투자** | GrubMarket $50M(밸류 $3.5B+), TRACT €18.6M 시리즈A, 미트박스 코스닥 상장, 마켓보로 CJ 등 투자 |
