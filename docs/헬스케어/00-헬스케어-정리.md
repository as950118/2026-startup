# AI 헬스케어·웨어러블 정리

> **목적**: AI 기반 헬스케어·웨어러블 방향 정리. 낙상·스마트옷 등 기존 제품을 피하고, 경쟁력을 가질 수 있는 분야·플랜만 모음.  
> **출처**: 소액-아이템-정리, 2026-분야별-경쟁력-아이템-정리, CB-Insights 조사

---

## 1. 방향 요약

| 구분 | 내용 |
|------|------|
| **타깃** | AI 기반 헬스케어 + 웨어러블 |
| **피할 것** | 낙상방지센서, 스마트 옷 등 이미 포화된 영역 |
| **전략** | 소액(1억 이하)으로 시작 → 니치 1개 PMF → 구독·SaaS 결합 후 스케일업 |

---

## 2. 경쟁력을 가질 수 있는 분야

| 분야 | 설명 | 포지션 |
|------|------|--------|
| **욕창·체압 관리** | 침대/의자용 압력 센서 매트 + AI로 체위·체압 분석, 욕창 위험 예측·알림 | 요양원·병원 B2B 니치 |
| **헬스케어 음성 AI + 웨어러블** | 웨어러블 데이터 + 음성 기록·문의·예약·상담. 다국어·접근성 | CB Insights "Voice AI to the test" 연계 |
| **실시간 모니터링 + AI 인사이트** | 단순 수치가 아닌 이상 징후·트렌드·악화 위험 예측 → 알림·리포트 | B2B 요양원·병원·기업보건 |
| **엣지 AI + 웨어러블** | 데이터를 클라우드 대신 엣지에서 추론. 프라이버시·저지연 | Jetson·Coral 등으로 PoC |
| **니치 질환 집중** | 수면 질, 당뇨·혈당 연동, 재활·동작 품질 등 **한 가지** 문제에 특화 | 범용 밴드/옷과 구분 |
| **언번들 헬스 연동** | 개인/팀 맞춤 플랜, 디지털 헬스·멘탈·예방과 웨어러블 데이터 연동. HR·보험사 연동 | CB Insights "Unbundling employer health" |

---

## 3. 피할 것·차별화 포인트

- **피할 것**: 낙상방지센서, 스마트 옷(이미 다수 존재).
- **차별화**: 같은 “센서/웨어러블”이 아니라 **니치(B2B·욕창·수면·재활 등)** 또는 **AI 인사이트/음성/엣지**에서 구분.

---

## 4. 플랜 요약 (작게 시작 → 스케일업)

| 단계 | 기간(참고) | 목표 |
|------|------------|------|
| **0. 검증** | 1~2개월 | 니치 1개·가설 1줄·미니 인터뷰 5~10명 |
| **1. PoC/MVP** | 3~6개월 | 시제품 + 1곳 실사용 4주+, 피드백 |
| **2. PMF** | 6~12개월 | 구독 포함 반복 매출, 2~5곳 유료, 단위 경제 |
| **3. 스케일업** | 12개월~ | 채널/제품/지역 중 하나씩 확장 |
