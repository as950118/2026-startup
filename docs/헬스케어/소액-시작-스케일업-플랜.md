# 작게 시작해서 스케일업하는 플랜

> **목적**: AI 헬스케어·웨어러블 등 소액(1억 이하)으로 시작 → PMF 검증 → 스케일업 단계별 로드맵  
> **참고**: [소액-아이템-정리.md](../소액-아이템-정리.md) 비용·지원·전략 반영

---

## 1. 전체 흐름 (4단계)

| 단계 | 목표 | 기간(참고) | 투자 규모 | 성공 기준 |
|------|------|------------|-----------|-----------|
| **0. 검증** | 문제·타깃·가설 확정 | 1~2개월 | 비용 최소 | 1개 니치·1개 가설 문서화 |
| **1. PoC/MVP** | 동작하는 시제품 + 1곳 실사용 | 3~6개월 | 1천~3천만 원 | 1곳 이상 실제 사용·피드백 |
| **2. PMF** | 반복 가능한 매출·구독 | 6~12개월 | 3천만~1억 원 | 월 매출·갱신률·NPS 등 정량화 |
| **3. 스케일업** | 채널·제품·지역 확장 | 12개월~ | 1억+ 또는 수익 재투자 | MRR/ARR 목표, 단위 경제 확보 |

---

## 2. Phase 0: 검증 (작게 시작의 시작)

**목표**: "누구의 어떤 문제를, 어떤 제품/서비스로 푸는가"를 한 줄로 말할 수 있게 하기.

| 할 일 | 내용 |
|-------|------|
| **니치 1개 고정** | 예: 요양원 5~20병상, 또는 기업보건 소규모 사업장, 또는 수면/욕창/재활 중 **하나**만 선택 |
| **가설 문서화** | "○○가 △△ 때문에 □□ 불편하다 → 우리는 ◇◇로 □□를 줄인다" 형태로 1페이지 |
| **경쟁·기존 대안** | 낙상센서·스마트옷 등 기존 제품과 우리 포지션 차이 정리 (같은 건 피하고, 갭만 공략) |
| **미니 인터뷰** | 타깃 5~10명(시설장·간호사·구매담당 등)에게 "지금 뭐가 가장 불편한지" 질문만 해도 됨 |

**비용**: 거의 0원 (시간 위주).  
**다음 단계**: 가설이 "이 니치·이 문제에 집중하자"로 수렴되면 Phase 1 진입.

---

## 3. Phase 1: PoC / MVP (최소 비용으로 "동작하는 것" 만들기)

**목표**: 하드웨어+SW 조합이 **한 곳**에서 실제로 쓰이고, 피드백을 받을 수 있는 수준.

| 항목 | 전략 (소액 유지) |
|------|------------------|
| **하드웨어** | 라즈베리파이·ESP32·Jetson Orin Nano·기성 센서 모듈로 PoC → 커스텀 PCB는 검증 후. [소액-아이템-정리.md §2.3, §3] |
| **양** | 5~20대 수준 시제품으로 충분. OEM·소량 조립 파트너로 1차 납품. |
| **SW** | 앱·대시보드·알림은 MVP 범위만 (1~2개 핵심 기능). No-code/Low-code로 빠르게 만든 뒤, 필요 시 전환. |
| **1곳 실사용** | 요양원 1곳, 또는 기업 1곳과 **파일럿/무상/저가** 계약. "실제로 매일 쓰이고, 불편한 점을 말해준다"가 목표. |
| **정부 지원** | **예비창업패키지**(시제품 비용 30~40%), **초기창업패키지**(평균 7천만·최대 1억) 등 K-Startup 공고 확인. [§7] |

**예시 비용**: 시제품 500만~2천만 원 + 개발 1천만 원대 + 운영·소통 500만 원대 = **2천만~3천만 원** (지원금 반영 시 실질 부담 감소).

**성공 기준**:  
- 1곳 이상에서 **연속 4주 이상** 사용  
- "다음에 무엇을 바꾸면 좋을지" 리스트가 5개 이상 도출  

**다음 단계**: "이걸 돈 주고 쓸 만하다"는 신호가 나오면 Phase 2(PMF)로.

---

## 4. Phase 2: PMF (반복 가능한 매출·구독)

**목표**: 하드웨어 단매가가 아닌 **구독·데이터·관리 서비스**로 매출이 나고, 같은 니치에서 2~3곳 더 팔 수 있는지 확인.

| 항목 | 전략 |
|------|------|
| **수익 모델** | 디바이스 + **월/연 구독**(알림·대시보드·리포트·관리) 조합. [§6 소액 하드웨어 창업 전략] |
| **타깃 유지** | Phase 0~1에서 정한 **니치 그대로** (예: 소규모 요양원만). 다른 세그먼트는 PMF 후에. |
| **소량 양산** | 100~500대 수준. OEM·빌드잇·큐레몬·누코드 등 [§3] 활용해 인증·양산 비용 절감. |
| **인증** | KC·CE 등 **필수만 단계별** 진행. 의료기기는 "웰니스/건강관리" 포지션 유지 후, 검증되면 의료기기 전환 검토. [§9] |
| **정부 지원** | **디딤돌**(2억 이내 75%), **제품화 올인원팩**(최대 5천만), **소재·부품·장비 스타트업 100**(최대 2억) 등 [§7] 검토. |

**성공 기준 (정량 예시)**  
- 유료 고객 **2~5곳**  
- 월 구독 갱신률·NPS·"다음 시설 추천 의향" 등으로 **반복 매출 가능성** 확인  
- 단위 경제: (연간 매출 − 디바이스·운영 비용) / 고객 수가 양수이고, 고객 1곳당 LTV가 CAC 대비 2~3배 이상 나오는 추세  

**다음 단계**: 같은 니치에서 "같은 말로, 같은 채널로" 2~3곳 더 계약되면 Phase 3(스케일업) 준비.

---

## 5. Phase 3: 스케일업 (채널·제품·지역 확장)

**목표**: PMF된 1개 니치를 기반으로 **매출·고객 수·제품 라인**을 키우기.

| 축 | 확장 옵션 |
|----|-----------|
| **채널** | 동일 니치 내 더 많은 시설(요양원·병원·기업) 공략; 지자체·협회·배포 파트너 도입 |
| **제품** | 1개 디바이스 → 번들(센서+게이트웨이+앱) 또는 2번째 유스케이스(예: 욕창 → 수면/재활) |
| **지역** | 국내 타 지역 → 해외 1개국(규제·인증 정리 후) |
| **수직** | B2B 시설 단위 → 보험·HR·헬스 플랫폼 연동(언번들 헬스 등)으로 단가·LTV 상승 |

**재원**  
- Phase 2에서의 **수익 재투자**  
- **디딤돌·스타트업 100·초격차 스타트업** 등 1억 이상 규모 지원  
- 필요 시 시드·시리즈 A (PMF·단위 경제 수치로 설득)

**주의**  
- 니치를 하나씩만 확장 (한 번에 채널·제품·지역 다 넓히지 않기)  
- 스케일업 단계에서도 **구독·SaaS 비중**을 키워서 하드웨어 단매가 의존도를 낮추기 [§6]

---

## 6. 단계별 체크리스트 (한 페이지 요약)

| Phase | 해야 할 것 | 피해야 할 것 |
|-------|------------|--------------|
| **0. 검증** | 니치 1개, 가설 1줄, 미니 인터뷰 5~10명 | 여러 니치 동시 공략, "일단 만들어보자"만 반복 |
| **1. PoC/MVP** | 기성 보드·센서로 시제품, 1곳 실사용 4주+, 지원금 신청 | 커스텀 PCB·완제품 품질을 처음부터 추구 |
| **2. PMF** | 구독 포함 수익 모델, 2~5곳 유료, 단위 경제 확인 | 니치 바꾸기, 의료기기 인증을 전제로 한 초기 포지션 |
| **3. 스케일업** | 채널/제품/지역 중 하나씩 확장, 수익·지원금 재투자 | 동시에 채널·제품·지역 다 확장 |

## 7. 연도별 가시적 마일스톤 (예시)

| 시점 | 목표 (예시) |
|------|-------------|
| **0~3개월** | Phase 0 완료: 니치·가설 문서, 미니 인터뷰 5명 이상 |
| **3~6개월** | Phase 1 완료: MVP 시제품, 1곳 파일럿 4주+ 사용, 피드백 정리 |
| **6~12개월** | Phase 2 진행: 유료 2~5곳, 월 구독 매출 발생, KC 등 필수 인증 1차 완료 |
| **12~18개월** | PMF 정량 확정: 갱신률·NPS·LTV/CAC 추세 정리, 스케일업 축 결정(채널 vs 제품 vs 지역) |
| **18~24개월~** | Phase 3: 선택한 1개 축으로 확장(예: 요양원 20곳, 또는 보험/플랫폼 1곳 연동) |
