# 03. Business Strategy: Data-First to Robot-Ready

> **전략 핵심**: 2030년 휴머노이드 시대를 대비해, 지금부터 **데이터(Data)와 사용자 신뢰(Trust)**를 선점하는 **"Top-Down"** 접근법.

## 1. Phased Roadmap (단계별 전략)

### Phase 1: The Bio-Companion (2025 ~ 2027)
*   **형태**: 모바일 앱 + 웨어러블(워치/링) 연동 + 스마트홈 스피커.
*   **핵심 기능**:
    *   **Bio-Sync**: 심박, 수면, 활동량 데이터를 실시간 분석하여 감정/건강 케어 코칭.
    *   **IoT Control**: "잘 자요" 한마디에 조명, 온도를 조절하고 수면 유도 음악 재생.
*   **BM**: 구독 모델 (Premium Care), 헬스케어 데이터 리포트 판매(B2B).

### Phase 2: The Stationary Agent (2027 ~ 2029)
*   **형태**: 전용 디스플레이/카메라가 탑재된 탁상형 AI (또는 태블릿 키오스크).
*   **핵심 기능**:
    *   **Vision AI**: 사용자의 표정, 행동 패턴 인식 (넘어짐 감지, 우울한 표정 포착).
    *   **Pre-Embodied Interaction**: 로봇의 '눈'과 '귀' 역할을 미리 학습.
*   **BM**: 하드웨어 판매/임대, 실버케어 시설 B2B 공급.

### Phase 3: The Humanoid Soul (2030+)
*   **형태**: 휴머노이드 로봇 탑재형 소프트웨어 (OS / Module).
*   **핵심 기능**:
    *   **Transfer Learning**: Phase 1, 2에서 축적된 사용자의 모든 데이터(습관, 선호, 건강 이력)를 로봇에 이식.
    *   **"처음 봤지만 나를 가장 잘 아는 로봇"**: 로봇 구매 첫날부터 내 평생의 파트너처럼 행동.
*   **BM**: 로봇 제조사 대상 라이선싱 (OS Licensing), 데이터 마이그레이션 비용.

## 2. Pre-emptive Data Collection Strategy (선제적 데이터 수집)
경쟁력의 핵심은 **"로봇이 걷기 시작할 때, 우리는 이미 달리고 있다"**는 것입니다.

| 데이터 유형 | 수집 방법 (Phase 1~2) | 활용 (Phase 3: Humanoid) |
| :--- | :--- | :--- |
| **Bio-Signal** | 스마트워치/링 연동 (HR, Sleep, Activity) | 로봇이 사용자의 피로도를 예측하여 가사 노동 강도 조절 |
| **Space Context** | IoT 센서, 카메라 (Smart Home) | 로봇의 자율주행 및 객체 인식을 위한 가정 내 지도(Map) 사전 구축 |
| **Emotional Interaction** | 음성 대화, 텍스트 로그 분석 | 사용자의 성향(MBTI 등), 화법, 감정 트리거에 맞춘 대화 스타일 최적화 |
| **Routine Pattern** | 기상/취침 시간, 식사 패턴 기록 | 로봇의 일과표 자동 생성 (예: 아침 7시 기상 시 커피 대령) |

## 3. IoT Integration Strategy
*   **Hub 역할**: 별도 센서 설치 없이, 기존에 보급된 삼성 SmartThings, LG ThinQ, Google Home API와 연동.
*   **능동적 개입**: 사용자가 명령하기 전에 환경을 제어. (예: 수면 효율이 떨어지는 날, 다음날 아침 알람을 부드러운 음악으로 변경하고 커튼을 천천히 엶)
