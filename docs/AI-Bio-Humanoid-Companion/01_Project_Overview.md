# 01. Project Overview: From Digital Soul to Humanoid Brain

> **비전**: "현재는 당신의 건강과 마음을 돌보는 AI 컴패니언, 미래에는 휴머노이드 로봇의 두뇌(OS)."

## 1. 프로젝트 정의
이 프로젝트는 단순한 대화형 챗봇을 넘어, 사용자의 **생체 데이터(Bio)**와 **생활 환경(IoT)**을 통합하여 **초개인화된 헬스케어/멘탈케어**를 제공하는 AI 컴패니언을 개발하는 것을 목표로 합니다.
궁극적으로는 2030년 상용화될 **휴머노이드 로봇**에 탑재될 **"감성 지능 및 케어 특화 두뇌"**를 선점하는 것이 핵심 전략입니다.

## 2. 왜 지금인가? (Why Now?)
*   **Bio/Healthcare**: 스마트워치, 링 등 웨어러블 기기의 대중화로 실시간 생체 데이터 확보가 용이해짐.
*   **IoT Synergy**: 스마트홈 기기들이 보편화되면서 사용자의 생활 패턴 데이터 확보 가능.
*   **Humanoid Pre-game**: 휴머노이드 하드웨어는 테슬라 등이 해결하겠지만, 그 안에 들어갈 **"인간을 깊이 이해하고 돌보는 소프트웨어"**는 아직 지배적 강자가 없음.

## 3. 핵심 가치 제안 (Value Proposition)
1.  **Bio-Sync Care**: "오늘 좀 우울해 보이네"가 아니라, **"심박변이도(HRV)와 수면 데이터를 보니 스트레스가 높군요. 명상을 도와드릴까요?"**라고 접근.
2.  **Invisible Data Collection**: 사용자가 의식하지 않아도 일상 대화와 IoT 상호작용 속에서 **미래 로봇을 위한 학습 데이터(행동 패턴, 감정 반응)**를 선제적으로 수집.
3.  **Continuous Companion**: 스마트폰 -> 스마트홈 스피커 -> (미래) 휴머노이드 로봇으로 이어지는 **끊김 없는(Seamless) 관계** 구축.

## 4. 로드맵 요약
*   **Step 1 (Digital Companion)**: 모바일/PC 기반. 웨어러블/IoT 연동 헬스케어 코칭.
*   **Step 2 (IoT Hub Agent)**: 집안의 IoT 기기를 제어하며 공간 정보를 학습하는 에이전트.
*   **Step 3 (Humanoid OS)**: 축적된 데이터와 페르소나를 휴머노이드 로봇에 이식.
