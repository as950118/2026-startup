## 0. 실행용 후보 아이템(니치 1개 선택)

> 실제 진입 시 **반드시 1개 니치만** 선택 후 0단계 검증 → PoC → PMF 순서 진행.

| 후보 | 1차 타깃 | 포지션 | 다음 액션 |
|------|-----------|--------|-----------|
| **고령·돌봄 AI 컴패니언** | 요양시설·재가돌봄·B2G | 감정·인지 자극·가족 연동·건강 리마인더, “치료 아님” | 가설 1페이지 → 시설/가족 미니 인터뷰 5~10명 |
| **멘탈헬스 지원(니치)** | 기업복지·대학·청년 | CBT·성찰·일상 지원, 치료 대체 금지·가드레일 | 기업 HR·대학 상담센터 인터뷰, 규제(의료광고 등) 사전 확인 |
| **B2B 고객참여 에이전트** | SMB·콜센터·영업팀 | Zoom·Dynamics 보조 또는 니치 채널·산업 특화 | 1개 채널(예: 음성·챗) 1개 산업 PoC, 단위 경제 검증 |
| **교육·튜터링 컴패니언** | EdTech·기업교육 | 개인화 학습·퀴즈·맥락, 교사 보조 | 1과목·1타깃 MVP, 보조 역할 명확화 |
| **멀티모달·엣지 컴패니언** | 프라이버시·저지연 요구 고객 | 온디바이스 추론·텍스트+음성+표정 | 기술 PoC(엣지 LLM·멀티모달), 파트너(칩·OS) 검토 |

**No-Go 시**: 다른 니치로 피벗 또는 §12 분기별 Go/No-Go에 따라 종료·재검토.

## 1. 사업 개요 및 비전

### 1.1 사업 정의

| 항목 | 내용 |
|------|------|
| **사업명(가칭)** | AI 컴패니언 니치 솔루션 (B2C 또는 B2B 1개 PMF 후 확장) |
| **한 줄 정의** | 감정·동반·생산성 중 **한 가지** 니치(고령·멘탈·B2B·교육·멀티모달 등)에서 PMF를 달성하고, 구독·SaaS·라이선스로 반복 매출을 만드는 사업 |
| **타깃** | 니치별 상이 — 시니어·기업복지·SMB·EdTech·프라이버시 민감 기관 등 |
| **피할 것** | 범용 챗봇 포지션; “치료 대체” 주장; 청소년·취약계층 무가드레일; 규제 무시 |

### 1.2 비전·목표(3년)

| 시점 | 목표 |
|------|------|
| **1년** | 니치 1개 PMF, 유료 고객 또는 MAU/구독 수 목표, 단위 경제·리텐션 검증 |
| **2년** | 동일 니치 내 확대 또는 2번째 유스케이스 1개, LTV/CAC·NPS 개선 |
| **3년** | 채널·제품·지역 중 1개 축 스케일업, ARR 목표 설정 |

### 1.3 전략 원칙

- **니치 1개 우선** — PMF 전까지 한 세그먼트만 집중.
- **구독·SaaS 결합** — 일회성 앱/라이선스보다 월/연 구독으로 LTV 확대.
- **차별화** — 감정·행동 과학·도메인 전문성·가드레일·규제 선제 대응.
- **리텐션** — 30일 이탈 87% 대비 개인화·참여 설계 필수.

## 2. 사업 아이템별 상세

### 2.1 고령·돌봄 AI 컴패니언

| 구분 | 내용 |
|------|------|
| **제품** | 음성·텍스트(선택 멀티모달) AI 동반자. 감정 지원·인지 자극·일상 리마인더·가족/돌봄사 연동 |
| **타깃** | 요양시설·재가돌봄·B2G(지자체)·가족 구매 |
| **기술** | STT/TTS·LLM·대화 설계, 필요 시 엣지 추론·프라이버시 |
| **수익** | 시설/가구당 월 구독, 디바이스 번들(선택) |
| **실현** | 일본 ElliQ·BOCCO, 한국 Hyodol 참고; 현지 파트너·규제(의료·개인정보) 확인 |

### 2.2 멘탈헬스 지원(니치)

| 구분 | 내용 |
|------|------|
| **제품** | CBT·성찰·일상 지원·리마인더. “치료 대체 아님” 명시, 위기 시 전문기관 연계 |
| **타깃** | 기업복지·대학·청년·보험사 연동 |
| **기술** | LLM·가드레일·감정 분류·에스컬레이션 로직 |
| **수익** | B2B 연간/인당 구독, B2C 월 구독 |
| **실현** | Robyn·Woebot·Tell Me 패턴; 의료광고·심리치료 관련 규제 사전 검토 |

### 2.3 B2B 고객참여 에이전트

| 구분 | 내용 |
|------|------|
| **제품** | 상담·영업·마케팅 중 1개 채널 특화 에이전트. CRM·미팅 툴 연동 |
| **타깃** | SMB·콜센터·영업팀, 1개 산업(예: 금융·이커머스) |
| **기술** | LLM·RAG·API(CRM·커뮤니케이션), 음성 채널(선택) |
| **수익** | 시트·에이전트 수·호출당 과금, 연간 라이선스 |
| **실현** | Zoom·Dynamics·Cognism와 차별화(니치 산업·채널·가격) 필요 |

### 2.4 교육·튜터링 컴패니언

| 구분 | 내용 |
|------|------|
| **제품** | 과목·연령 1개에 특화. 개인화 학습·퀴즈·맥락 제공, 교사 보조 |
| **타깃** | EdTech·학교·기업교육 |
| **기술** | LLM·RAG(교재)·적응형 난이도·진도 추적 |
| **수익** | 학교/기관 연간, 또는 학습자 월 구독 |
| **실현** | Google Learn About·MeluBot 참고; 교육 규제·저작권 고려 |

### 2.5 멀티모달·엣지 컴패니언

| 구분 | 내용 |
|------|------|
| **제품** | 텍스트+음성+표정(선택)·온디바이스 추론. 프라이버시·저지연 |
| **타깃** | 헬스·금융·공공·프라이버시 민감 B2B |
| **기술** | 경량 LLM·엣지 런타임(Jetson·Coral 등)·멀티모달 파이프라인 |
| **수익** | 라이선스·구독·하드웨어 번들 |
| **실현** | PoC 비용·파트너(칩·디바이스) 확보가 관건 |

## 3. 수익 모델 요약

| 유형 | 예시 |
|------|------|
| **B2C 구독** | 월 $10~20, 프리미엄(연인·멘토 등 역할 잠금 해제) |
| **B2B 구독/라이선스** | 시설·좌석·에이전트 수·API 호출당, 연간 계약 |
| **B2G** | 지자체·요양 정책 연계, 인당/시설당 연간 |
| **하이브리드** | 디바이스 1회 + 월 구독(앱·관리·업데이트) |

## 4. 실현 가능성 체크

| 항목 | 판단 | 비고 |
|------|------|------|
| **시장** | 유리 | 30%+ CAGR, 사회적 교류·멘탈·B2B 수요 증가 |
| **경쟁** | 니치 필수 | Replika·Character·Xiaoice·Zoom 등과 직접 겹치지 않는 세그먼트·포지션 |
| **규제** | 주의 | FTC·주법·EU AI Act·의료·청소년 보호; 가드레일·투명성 설계 |
| **윤리** | 필수 | 의존·중독·취약층 보호; “치료 대체” 금지, 에스컬레이션 |
| **리텐션** | 도전 | 87% 30일 이탈 — 개인화·습관·감정 설계로 극복 필요 |
| **기술** | 가능 | LLM·STT/TTS·API 연동은 상용화 수준; 엣지·멀티모달은 PoC 단계 |

## 5. 분기별 Go/No-Go(참고)

| 시점 | 확인 항목 |
|------|-----------|
| **Q1** | 니치 가설·타깃 리스트·미니 인터뷰 5~10명, 규제 1차 검토 |
| **Q2** | PoC 범위·자원·파트너, 1차 PoC 결과 |
| **Q3** | 유료 파일럿·단위 경제·리텐션 지표 |
| **Q4** | PMF 판단·스케일업 방향(채널/제품/지역) |
