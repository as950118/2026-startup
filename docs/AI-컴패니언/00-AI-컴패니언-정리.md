## 1. 방향 요약

| 구분 | 내용 |
|------|------|
| **정의** | 관계 유지·감정 지원·생산성 보조를 목적으로 하는 AI 시스템(텍스트/음성/멀티모달·로봇 포함) |
| **시장 규모** | 2024년 약 281억 달러 ~ 2025년 371억 달러 수준, 2030년 1,400억 달러·2035년 5,524억 달러 전망(CAGR 30~31%) |
| **전략** | B2C(감정·동반자) vs B2B(고객참여·생산성)·헬스(멘탈·고령)·교육 등 **니치 1개** PMF 후 확장 |

| **니치 세그먼트** | 시니어 전용, 청소년 가드레일, 특정 언어·문화 맞춤(예: 아랍어 Allam) | 지역·연령·규제 맞춤 |

## 3. 피할 것·차별화 포인트

- **피할 것**: 범용 챗봇과 동일시; “치료 대체” 주장(규제·윤리 리스크); 청소년·취약계층 무가드레일 배포.
- **차별화**: **감정·행동 과학 + 습관 형성 + 장기 개인화 데이터**로 “변화” 유도(단순 작업 완료가 아님); 도메인 전문성(의료·교육 배경); 규제·윤리 선제 대응(EU AI Act, FTC, 주법).

## 4. 시장·트렌드 요약

| 항목 | 내용 |
|------|------|
| **앱 성장** | 2022~2025년 컴패니언 앱 700% 성장; 5억+ 다운로드, Character.AI만 월간 2천만(절반 24세 미만) |
| **타입** | 텍스트 기반이 점유율 1위(43~44%); 멀티모달이 CAGR 최고 성장 |
| **애플리케이션** | 사회적 교류·동반자 > 멘탈헬스 지원 > 교육·개인 비서 |
| **산업** | Consumer > Businesses > Healthcare > Education |
| **지역** | 북미 점유율 1위(34% 내외); 아시아태평양이 최고 CAGR |

## 5. 규제·윤리

- **미국**: FTC AI 컴패니언 광고·안전·데이터 조사; 뉴욕 RAISE Act·AI 컴패니언 안전(2025.11); 주별 패치워크(콜로라도·캘리포니아·유타 등).
- **EU**: AI Act — 기만·취약성 악용 금지, 투명성·리스크 관리·사고 보고, 감정 의존·미성년자 보호 반영.
- **윤리**: 의존·중독, 사회성 약화, 취약 사용자 사고(청소년 등) 대비 가드레일·윤리 설계 필수.
